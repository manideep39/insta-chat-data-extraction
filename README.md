# insta-chat-data-extraction

### How to use:

- Clone the repo
- Do `npm install`
- Unzip the `inbox.zip` and add `inbox` folder to the root folder.
- Run the script with, `node index.mjs`
- You will have the `output.json` file in the root directory.
